## how to use

```sh
# install dependence
pnpm i

# edit data/resume.js and replace avatar image with your owen

# preview in broswer
pnpm dev

# click the button to print to pdf
```

## want to custom theme color

open browser devtools (F12), and edit css variable `--color-primary`
